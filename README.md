## 📖 Sobre a Disciplina
Este laboratório tem como objetivo principal a implementação prática de conceitos fundamentais da ciência da computação. Aqui, a teoria se transforma em código, focando na análise de complexidade, alocação de memória, organização e manipulação eficiente de dados.
